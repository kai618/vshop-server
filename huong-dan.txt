TẠO SERVER NodeJS ĐƠN GIẢN
+ Tạo thư mục cho server project
+ Chạy lệnh để cấu hình ứng dụng server:
	npm init
+ Cài đặt express module hỗ trợ môi trường server trong nodejs
	npm install express --save

+ Tạo file server.js  trong thư mục project
	- import express với Javascript
		const express = require('express');
	Hoặc:
	- import express  nếu dùng với Typescript
		import * as express from 'express'
+ Start server 
	node server.js
hoặc 
	npm start
	
+ Cài thêm CORS là middleware dùng để thông dịch mỗi request tới và add thêm corresponse header cho mỗi Respones
Corresponse header này xác định url nào được chấp nhận hoặc thông báo mã response tương ứng.
	npm install cors --save
+ Thêm đoạn code sử dụng cors sau vào file server.js ta được file server.js demo đơn giản như sau:
	const cors = require('cors')

	var corsOptions = {
	  origin: 'http://localhost:4200',
	  //domain được phép gọi request mà server chấp nhận (vd: request đến từ http://localhost:4200  được server cho phép), 
	  //giả sử node server là http://localhost:8000
	  optionsSuccessStatus: 200 // some legacy browsers (IE11, various SmartTVs) choke on 204 
	};
	const app = express();
	app.use(cors(corsOptions));
	
	
	app.listen(8000, () => {
		console.log('Server started!');
	  });
	app.route('/api/items').get((req, res) => {
	  console.log('items');
		res.send([{ name: 'lilly', id:'id1' }, { name: 'Oscar', id:'id2' }]
		);
	});

TẠO SERVICE PHÍA CLIENT VÀ GỌI SERVICES TỪ NODEJS SERVER
	+ Tạo service tên "items" bằng lệnh sau: "ng generate service items" (hoặc "ng g s items")
		...
		...//tự import các modules còn thiếu cho đoạn code bên dưới
		...
		export class ItemsService {

		  constructor(private http: HttpClient) { }

		  getItems():Observable <Item[]>{
			return this.http.get<Item[]>('http://localhost:8000/api/items/');

		  }
		}
		Lưu ý: Để sử dụng được HttpClient thì trong file app.module.ts phải import HttpClientModule vào như sau:
				
				import { HttpClientModule } from '@angular/common/http';
				...
				@NgModule({
				  declarations: [
					AppComponent,
					...
				  ],
				  imports: [
					
					HttpClientModule,//nơi cần bổ sung
					BrowserModule,
					AppRoutingModule
				  ],
				...
	+ Trong component muốn sử dụng ItemsService cần bổ sung nội dung sau:
		//bổ sung vào file .ts
			...
			...//tự import các modules còn thiếu cho đoạn code bên dưới
			
			constructor(private service: ItemsService) { }
			
			itemList: Observable<Item[]>;
			ngOnInit() {
				this.itemList = this.service.getItems();
			}
			
		//tự viết lệnh hiển thị danh sách itemList trên file html của component
		//(tương tự hiển thị danh sách items khi lấy dữ liệu từ firebase
	+ add selector của component này vào AppComponent
